Loudwater Weather
Welcome to Loudwater Weather, the local weather station for Loudwater. At the moment, this website is still in production and you could see a few changes every now and then.

The weather station is using a Raspberry Pi Pico 2 WH and an AM2320 sensor to measure temperature and humdity. The time and date is done using javascript.

Hardware
